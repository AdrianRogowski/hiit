## Purpose

Primary interactive element for actions like Start, Pause, Resume, Stop, Skip.

## Variants

- **Primary**: Main action (Start Timer) - full width, prominent
- **Secondary**: Supporting actions - outlined style
- **Control**: Timer controls (Pause, Skip, Stop) - icon + text
- **Danger**: Destructive actions (Stop) - `color-error` tint

## Sizes

- **Large**: Primary CTA buttons
- **Medium**: Standard controls
- **Icon**: Icon-only buttons

## States

- Default
- Hover (slight glow)
- Active (pressed)
- Disabled (reduced opacity)
