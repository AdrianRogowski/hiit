## Purpose

Overlay dialog for multi-device sharing, confirmations, and settings.

## Features

- Backdrop overlay (dimmed background)
- Centered content panel
- Close button
- Keyboard dismissable (Escape)
- Focus trap for accessibility

## Variants

- **Default**: Standard content modal
- **Confirmation**: Yes/No action modal
- **Share**: Multi-device sharing modal
